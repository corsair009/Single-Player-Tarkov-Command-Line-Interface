Single Player Tarkov Command Line Interface is a python-based tool that allows users to instantly look up useful information, instead of navigating through the wiki.
Currently made to be used with the mod Single Player Tarkov for Escape From Tarkov, but it can be used for EFT.

INSTALLATION: copy and paste the code in either VScode, the official python IDE, any browser based IDEs, or any others.
I am currently working on how to get it to work with Windows Powershell.

DISCLAIMER! be sure to use pip install termcolor on powershell/vscode/your ide of choice, or SPTCLI will not work and give an error.
The color will also not show on some IDEs even if you install termcolor, but it will still be functional.
